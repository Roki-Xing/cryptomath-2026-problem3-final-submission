# Official reference boundary

This directory records the official-source boundary used by
`docs/OFFICIAL_SPEC_INTERPRETATION.md`.

The official challenge landing page is public. The detailed analysis PDF is not
checked into this repository because redistribution rights are not established.
`SOURCES.json` records the public URL, retrieval timestamp, redistribution
status, and the page map used by the repository's official-spec interpretation.

If the official analysis PDF is later redistributable, add the PDF here and
update `SOURCES.json` with the verified filename and SHA-256.
